# 웹 검색 스킬

## 개요
인터넷에서 정보를 검색하고 결과를 요약해주는 스킬입니다.

## 사용 방법
사용자가 "~에 대해 검색해줘", "~가 뭐야?" 등의 질문을 하면 이 스킬을 활용합니다.

## 실행 단계
1. 사용자 질문에서 검색 키워드 추출
2. 검색 엔진 API 호출 (Google, Bing 등)
3. 상위 5개 결과 수집
4. 핵심 내용 요약
5. 출처와 함께 답변 제공

## 응답 형식
```
📌 검색 결과: [키워드]

[요약 내용]

🔗 출처:
- [링크1]
- [링크2]
```

## 주의사항
- 최신 정보 우선
- 신뢰할 수 있는 출처만 사용
- 불확실한 정보는 명시
