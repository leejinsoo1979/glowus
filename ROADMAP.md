# 🗺️ GlowUS Evolution Roadmap

> **Target:** Building the First Operating System for Digital Lifeforms.

## 🟢 Phase 1: Spark (The Beginning)
**Goal:** 에이전트에게 '기억'과 '형체(Graph)'를 부여한다.

- [x] **Neural Map Core** (3D/2D Graph Visualization)
- [x] **Neural Search** (Semantic Knowledge Retrieval)
- [x] **Basic Agent Persona** (Amy, Rachel, Jeremy)
- [x] **Long-term Memory System** (Vector DB + Graph Linkage)
- [x] **Workspace Control Tower** (Initial Dashboard Interface)

## 🟡 Phase 2: Awareness (Self-Regulation)
**Goal:** 에이전트가 스스로 생각하고 검증하며 도구를 만든다.

- [ ] **Agent Tribunal System (청문회)**
    - [ ] Multi-Agent Debate Room (1:N Chat)
    - [ ] Auto-Moderator & Consensus Mechanism (합의 도출)
    - [ ] Cross-Verification Logic (Peer Review)
- [ ] **Self-Evolution CLI**
    - [ ] Agent Skill Shopping Cart UX
    - [ ] Dynamic Skill Coding (Python Script Generation)
    - [ ] Safety Sandbox for Self-Coded Skills
- [ ] **Memory Hygiene System**
    - [ ] Contamination Alert (오염 경보)
    - [ ] Partial Memory Dilution Logic (희석 프로세스)
    - [ ] Identity Reinforcement Ceremony (Genesis UX)

## 🔵 Phase 3: Civilization (The Society)
**Goal:** 인간의 개입을 최소화한 자율 협업 생태계.

- [ ] **Macro-Commit System**
    - [ ] "업무 지시"만 하면 알아서 수십 개의 커밋 생성
    - [ ] Human-as-Approver Workflow (결재 시스템)
- [ ] **Agent Social Graph**
    - [ ] 에이전트 간 친밀도 및 신뢰도 시각화
    - [ ] Team Dynamics Simulation
- [ ] **Dream Mode**
    - [ ] 유휴 시간(Idle Time) 동안의 기억 정리 및 강화(Consolidation) 시각화

## 🟣 Phase 4: Expansion (Work as Code)
**Goal:** 개발자를 넘어 모든 지식 근로자를 위한 OS로 확장.

- [ ] **Universal File Adapter** (Docs, PPT, Excel -> Neural Map)
- [ ] **Legacy Project Archeology** (기존 프로젝트 자동 분석 및 시각화)
- [ ] **Enterprise Control Center** (수천 명의 에이전트 관제)
